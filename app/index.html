Why Us?

We can give event organizers a peace of mind when managing events finally. All the hectic work of getting excel sheets , managing attendees and a lot more can be done at ease

Features

Event Organisers have the freedom to a lot of features like :

  • Create new event
  • Revoke an attendee
  • Issue refunds
  • Cancel an event
  • Watch Statistics like :
    • Area-wise attendees
    • Age-wise attendees
    • Category-wise attendees
